------------------ READ_ME ---------------------

- Application framework
	Angular 5
	.Net Core 2.1
- IDE used
	Visual Studio 2017
	
How to run the application
	- Solution file is included in the repository - TempretureConverter.sln
	- Open the solution in Visual Studio 2017
	- Build the Application to create npm folder for denpendent modules (this will take few minutes)
	- Run with IISExpress
	
Working
	- Open Tab Tempreture Converter
	- Enter value and focus out to calculate other 2 values (check video)
